Azure Monitor日志查询现在支持审计

最后，这里有一点Azure Monitor从2021年末开始的更新，你可能在假期错过了。

多年来，Azure Monitor的名称发生了很多变化(包括日志分析、OMS、监控解决方案)。随着时间的推移，我们看到了许多改进和功能的增加。今天，Azure Monitor已经成为一个强大的监控解决方案，并帮助您监控Azure内外的应用程序、操作系统和资源。

所以你可以捕捉很多很多的监控数据。但是你如何处理所有这些数据呢？你可以通过一个叫做日志查询的Azure Monitor特性来做到这一点。嗯，日志查询使用一种强大的查询语言来帮助你搜索和分析所有的日志数据。但是缺少的一点是跟踪和监控日志查询本身的能力。

因此，正如微软所说，有了这个更新，你现在可以跟踪“查询何时运行、由谁运行、使用了什么工具、查询文本以及描述查询执行的性能统计数据。”这确实可以帮助您提高日志查询的性能、和安全性。
